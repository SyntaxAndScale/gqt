# Product Requirement Document: Gqueues TUI (gqt)

## Overview
A terminal-based user interface (TUI) for managing Gqueues tasks, designed for efficiency and keyboard-centric workflows.

## Core Features
- **Three-Pane Layout:** A VSCode-inspired layout (Side Bar, Task List, Detail View).
- **Keyboard Navigation:** Full control via keyboard shortcuts.
- **Offline-First / CRDT:** Cache operations locally and sync with the Gqueues API, supporting robust conflict resolution.
- **CRUD Operations:** Create, Read, Update, and Delete tasks.

## Target Environment
- Primary: Linux (Terminal).

## User Personas
- Power users who prefer terminal workflows over web interfaces.
- Users requiring offline access to their task lists.
